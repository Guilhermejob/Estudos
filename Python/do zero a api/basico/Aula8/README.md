# Modulos, pacotes e importações


## Modulos
Um modulo em python é um arquivo de código que contem definições funções, classes, objetos, listas, os modulos nos permitem dividir nosso código em partes para facilitar na manutenção e reutilização de códigos em outras partes da aplicação

para criar um modulo basta criar um arquivo com a extenção .py


## Pacotes
Pacotes são uma coleção de módulos agrupados emuma estrutura de pastas para que um pacote seja reconhecido pelo python como tal é necessario criar um arquivo `__init__.py`  na pasta do pacote, o `__init__.py` não necessita conter códigos ou instruções, mas podem conter 

## Importações
As importações são um meio de trazer os códigos dos nossos modulos e pacotes para nossa aplicação principal consumir

### Importação Absoluta
A importação absoluta é um tipo de importação que nos permite importar modulos e pacotes apartir do caminho absoluto ou seja o caminho completo apartir da raiz do sistema

### Importação Relativa
A importação absoltuta nos permite importar apartir do caminho do módulo em relação ao script atual ou ao ambiente de trabalho atual, em vez de precisar usar o caminho absoluto completo do modulo
